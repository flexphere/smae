# Satori Mail Analyzer Extension

satori管理画面にて送信済みメールのデータ集計を行うプラグイン

## Installation

1. gitからソースをダウンロード
  ```
  git clone https://github.com/flexphere/smae.git
  ```
2. chromeにて[設定] > [拡張機能] を開き、画面右上の「デベロッパー モード」にチェックを入れる
3. （1.）でダウンロードされたフォルダをchromeの画面へDrag&Drop

## Usage

1. Satori管理画面へログイン
2. [アクション] > [メール] を開きchrome拡張の![アイコン](https://github.com/flexphere/smae/raw/master/images/icon/satori_32.png)アイコンをクリック
3. 集計期間<sup>*1</sup>を入力し、Analyzeをクリック
4. 件名別の集計が表示されます  

<sup>**1</sup>期間指定機能は実装予定（実装予定の機能については[こちら](https://github.com/flexphere/smae/projects/1)を参照）
